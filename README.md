# Ciência da Computação - UniCEUB

# 📖 Sobre o Projeto
Este repositório reúne conteúdos, atividades, exercícios e projetos desenvolvidos ao longo da graduação em Ciência da Computação no Centro Universitário de Brasília (UniCEUB).

O objetivo é centralizar materiais acadêmicos, práticas de programação, desafios propostos em disciplinas e projetos desenvolvidos durante a formação.

# 🎯 Objetivo
Organizar e documentar conteúdos relacionados ao curso de Ciência da Computação, promovendo aprendizado prático, desenvolvimento técnico e evolução contínua nas áreas da tecnologia e programação.

# 🛠️ Tecnologias e Ferramentas
Ao longo do curso, diferentes linguagens e ferramentas podem ser utilizadas, incluindo:

- C++
- Python
- Java
- Git & GitHub
- SQL
- VS Code
- Dev-C++
- Banco de Dados
